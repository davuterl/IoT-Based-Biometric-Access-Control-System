# Proje Hakkında

## Türkçe Açıklama

### Proje Özeti
Bu proje, Arduino ve Python kullanılarak geliştirilmiş bir sistemdir. Servo motorların kontrolü, yüz tanıma, Telegram bot entegrasyonu ve bir grafiksel kullanıcı arayüzü (GUI) içermektedir. Proje, farklı kullanıcı gruplarını tanıyarak servo motorları kontrol edebilmektedir. Kullanıcılar, Telegram botu üzerinden komut göndererek servo motorları hareket ettirebilir.

### Dosya ve Klasör Yapısı

1. **`arduino.ino`**:
   - Arduino kodlarını içerir.
   - Servo motorların kontrolü için `Servo` kütüphanesi kullanılmıştır.
   - Üç farklı servo motorun başlangıç pozisyonlarına ayarlanması ve seri iletişim başlatılması gibi işlemleri içerir.
   - Gelen komutlara göre servo motorları hareket ettirir.
   - Komutlar:
     - `1`: Grup 1 için Servo Motor 1 hareket eder.
     - `2`: Grup 2 için Servo Motor 2 hareket eder.
     - `3`: Grup 3 için Servo Motor 3 hareket eder.
     - Geçersiz komutlar için hata mesajı döner.

2. **`görünüt.py`**:
   - Python ile yazılmış ana uygulama dosyasıdır.
   - Telegram bot entegrasyonu, yüz tanıma, GUI oluşturma ve seri iletişim gibi işlevleri içerir.
   - Kullanıcı bilgileri ve Telegram bot ayarları bu dosyada tanımlanmıştır.
   - Telegram botu, kullanıcıların yüz tanıma ve motor kontrol işlemlerini uzaktan gerçekleştirmesine olanak tanır.
   - Yüz tanıma işlemi için `face_recognition` kütüphanesi kullanılmıştır.
   - GUI, `Tkinter` kütüphanesi ile oluşturulmuştur ve kullanıcı dostu bir arayüz sunar.

3. **`training_images/`**:
   - Yüz tanıma için kullanılan eğitim görüntülerini içerir.
   - Alt klasörler, her bir kullanıcıya ait görüntüleri barındırır.
   - Örnek klasör yapısı:
     - `1. grup/`: Davut Erol'un görüntüleri.
     - `2. grup/`: Ufuk Karan Soyoral'ın görüntüleri.
     - `3. grup/`: Anonim kullanıcı görüntüleri.

4. **`__pycache__/`**:
   - Python tarafından oluşturulan önbellek dosyalarını içerir.
   - Önbellek dosyaları, uygulamanın daha hızlı çalışmasını sağlar.

### Kullanılan Teknolojiler
- **Arduino**: Servo motor kontrolü.
- **Python**: Yüz tanıma, Telegram bot entegrasyonu ve GUI.
- **OpenCV**: Görüntü işleme.
- **Face Recognition**: Yüz tanıma.
- **Tkinter**: Grafiksel kullanıcı arayüzü.
- **PyTelegramBotAPI**: Telegram bot entegrasyonu.

## English Description

### Project Summary
This project is a system developed using Arduino and Python. It includes servo motor control, face recognition, Telegram bot integration, and a graphical user interface (GUI). The project can control servo motors by recognizing different user groups. Users can send commands via the Telegram bot to move the servo motors.

### File and Folder Structure

1. **`arduino.ino`**:
   - Contains Arduino code.
   - Uses the `Servo` library for controlling servo motors.
   - Includes operations such as setting the initial positions of three different servo motors and starting serial communication.
   - Moves servo motors based on received commands.
   - Commands:
     - `1`: Servo Motor 1 moves for Group 1.
     - `2`: Servo Motor 2 moves for Group 2.
     - `3`: Servo Motor 3 moves for Group 3.
     - Invalid commands return an error message.

2. **`görünüt.py`**:
   - The main application file written in Python.
   - Includes functionalities such as Telegram bot integration, face recognition, GUI creation, and serial communication.
   - User information and Telegram bot settings are defined in this file.
   - The Telegram bot allows users to perform face recognition and motor control remotely.
   - Face recognition is implemented using the `face_recognition` library.
   - The GUI is created using the `Tkinter` library and provides a user-friendly interface.

3. **`training_images/`**:
   - Contains training images used for face recognition.
   - Subfolders store images for each user.
   - Example folder structure:
     - `1. grup/`: Images of Davut Erol.
     - `2. grup/`: Images of Ufuk Karan Soyoral.
     - `3. grup/`: Images of anonymous users.

4. **`__pycache__/`**:
   - Contains cache files generated by Python.
   - Cache files improve the application's performance.

### Technologies Used
- **Arduino**: Servo motor control.
- **Python**: Face recognition, Telegram bot integration, and GUI.
- **OpenCV**: Image processing.
- **Face Recognition**: Face recognition.
- **Tkinter**: Graphical user interface.
- **PyTelegramBotAPI**: Telegram bot integration.
